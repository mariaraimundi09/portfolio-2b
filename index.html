Minha Rede Social: Uma Webpage Personalizada

O projeto é uma página web criada com HTML e CSS, é uma plataforma digital que permite a interação e comunicação entre pessoas. Nelas, usuários criam perfis para compartilhar conteúdo (textos, fotos, vídeos) e se conectar com outros, seja para fins pessoais ou profissionais.

A estrutura do site é baseada em cores acessiveis para o público, as cores amarelo claro e roxo/lilás são usadas de forma complementar para criar um design de aplicativo visualmente atraente, amigável e com uma sensação de leveza e alegria, ajudando na melhor visualizaçã do site .

O projeto foi criado com intuito da socialização entre donos de pets que querem comartilhar seus pets, para ter mais acesso a outros bixinhos para poder procurar por um animal que esteja em adoção e também poder conversar com donos de outros pets.

Projeto Robótica

O projeto consiste em um semáfaro que é ativado ao precionar um botão que dispara um timer,no intuito de cronometrar o o tempo entre as cores que irão apagar ou ascender conforme o tempo proposto entre elas, e irá sinalizar para os motoristas.

O projeto tem como composição uma versão diminuida de um semáfar, um relógio digital, uma placa protoboard, fios jumpers, uma placa arduino UNO e um botão, que irão fazer o processo descrito acima .

Foi produzido usando as aulas que estão disponiveis no classroom, colocamos o código no site, fizemos os ajustes necessários, retiramos códigos e adicionas alguns, e o projeto funcionou como esprado.

Led e Resistores

Este projeto foi reasizado com o intuito de aprendizagem dos alunos, para fazerem um sistema básico de robótica na prática.

Ao ligar o LED depois de executar os comandos da aula, os alunos conseguem ter uma base de como serão feios os próximos projeto.
